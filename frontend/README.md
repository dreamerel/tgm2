# Telegram Manager Frontend

## Запуск

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите приложение:
   ```bash
   npm start
   ```

## Страницы
- `/register` — регистрация
- `/login` — вход
- `/` — основной интерфейс (после входа)

## Backend
Backend должен быть запущен на http://localhost:5000
